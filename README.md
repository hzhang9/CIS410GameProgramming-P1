# **CIS410GameProgramming-P1**
## **Operation**
### For Win 64 system:  
1. Complete clone entire repository.
2. Run Rollaball\Builds\Rollaball.exe.
### For others system or WebGL:
1. Complete clone entire repository.  
2. Use unity editor to build the corresponding game files for different system.  
3. Run the game file.
